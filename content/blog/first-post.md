## Introduction

This is a sample post showcasing the **Modus theme** for Zola, ported from the Hugo theme by [goofansu](https://github.com/goofansu/hugo-modus).

## Features

The theme includes:

- Beautiful Modus Operandi (light) and Modus Vivendi (dark) color schemes
- Responsive design using Tailwind CSS
- Syntax highlighting support
- Clean and minimalist layout
