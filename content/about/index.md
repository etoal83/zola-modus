## About This Theme

The **Modus theme** is an opinionated Zola theme that brings the beautiful and accessible Modus color schemes to the Zola static site generator.

## Features

This theme is designed with the following principles in mind:

### Accessibility

- WCAG AAA compliant color contrast
- Readable typography optimized for long-form content
- Proper semantic HTML structure
- Keyboard navigation support

### Performance

- Minimal JavaScript (none by default)
- Optimized CSS with Tailwind
- Fast page loads
- Static site generation with Zola

### Design Philosophy

- **Minimalist**: Clean, distraction-free design
- **Readable**: Typography optimized for reading
- **Responsive**: Works beautifully on all screen sizes
- **Dark Mode**: Automatic theme switching based on system preferences

## Technology Stack

- **Zola**: Fast static site generator written in Rust
- **Tailwind CSS v4**: Modern utility-first CSS framework
- **Modus Colors**: Accessible color schemes by Protesilaos Stavrou
- **GitHub Pages**: Free and reliable hosting
